# メタ文字

「.」 改行を除く任意の一文字

# 文字種

「\d」 0-9
「\D」 \d以外
「\w」 a-z, A-Z, 0-9, _
「\W」 \w以外
「\s」 スペース, タブ, 改行, など
「\S」 \s以外

# 量指定子

「{n}」 n個
「{min, max}」 min以上、max以下
「{min,}」 min以上
「?」 0回以上、1回以下({0,1})
「+」 1回以上({1,})
「*」 0回以上({0,})

# アンカー

「^」(キャレット) 行の先頭
「$」 行の末尾
「\b」 単語の境界

# 選択子

「abc|123」 abc or 123

# 文字クラス

「[abc]」 a, b, cのどれか一文字

# 文字クラス内で使えるメタ文字

「^」(キャレット) 否定 ([の直後のみ有効)
「-」(ハイフン) 範囲 (例：A-C)

# 改行とタブ

「\t」 タブ
「\r\n」 改行(Windows)
「\r」 改行(v9までのmacOS)
「\n」 改行(Unix、v10以降のmacOS)
「\r\n|\r|\n」 改行(全てのOSで適応させたい場合)

# キャプチャ

()で囲って$1, $2,...と指定

# 後方参照

()で囲って\1, \2,...と指定
